About

Welcome to Local Events, a simple yet powerful web app designed and developed by Asish Pradhan. This platform helps communities stay informed about local gatherings, festivals, workshops, and activities happening nearby.

The idea behind Local Events came from a small-town perspective — where people often hear about local functions or celebrations only after they’ve happened. This app aims to bridge that gap by providing a central place to discover, share, and join upcoming community events.

Currently, users can:

  • View details about featured local events
  • Join events by submitting their details through a registration form
  • Create new event listings with title, location, and description

    • This project is built using HTML, CSS, and JavaScript on the frontend, and a lightweight Node.js + Express backend to handle user data submissions. All event and user details are safely stored in local JSON files for now.

    An Events Page is in development and will soon allow users to explore all upcoming and past events, making the experience more interactive and connected.

    Designed with simplicity and accessibility in mind, Local Events demonstrates how even small digital tools can bring communities closer together.
